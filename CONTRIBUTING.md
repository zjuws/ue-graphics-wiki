# 贡献指南

感谢你对UE图形渲染Wiki的兴趣！

## 如何贡献

### 报告问题

如果你发现文档中有错误或不清楚的地方，请：

1. 检查是否已有相关Issue
2. 创建新Issue，描述问题
3. 提供具体例子或参考资源

### 提交改进

1. **Fork** 本仓库
2. 创建特性分支 (`git checkout -b feature/amazing-feature`)
3. 提交更改 (`git commit -m 'Add amazing feature'`)
4. 推送到分支 (`git push origin feature/amazing-feature`)
5. 打开 **Pull Request**

## 文档标准

### 文件命名
- 使用数字前缀：`01-`, `02-`, 等
- 使用小写和连字符：`my-document.md`
- 中文文件名：`01-渲染管线.md`

### 文档结构
```markdown
# 标题

## 概述
简要介绍

## 核心概念
详细说明

## 代码示例
```cpp
// 代码
```

## 最佳实践
推荐做法

## 常见问题
FAQ

## 参考资源
链接
```

### 代码块
- 使用语言标记：```cpp, ```hlsl, ```python
- 添加注释说明
- 保持代码简洁

### 链接
- 使用相对路径：`[链接](./docs/other-doc.md)`
- 外部链接：`[官方文档](https://docs.unrealengine.com/)`

## 内容指南

### 准确性
- 基于官方文档
- 测试过的代码
- 引用可靠来源

### 清晰性
- 简洁的语言
- 逻辑清晰的结构
- 适当的代码示例

### 完整性
- 涵盖基础和高级
- 包含最佳实践
- 提供参考资源

## 审查流程

1. 自动检查（格式、链接）
2. 人工审查（内容、准确性）
3. 合并或请求修改

## 许可证

通过提交贡献，你同意你的贡献将在MIT许可证下发布。

## 问题？

- 创建Issue讨论
- 查看现有讨论
- 联系维护者

感谢你的贡献！🎉
